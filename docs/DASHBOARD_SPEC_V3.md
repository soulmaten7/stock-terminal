## 철학

1. **대시보드 = 긴 스크롤 페이지**. 2 viewport에 우겨넣지 않고, 위젯 콘텐츠 우선 크기. 5~6 viewport 내려가도 OK.
2. **TopNav 항목 ↔ 대시보드 위젯 1:1** (참고 사이트만 예외). 네비의 모든 메뉴는 대시보드 미리보기로 존재.
3. **위젯별 역할 명확화**. "대시보드 미리보기" vs "세부 페이지"는 정보 밀도 차이. 대시보드엔 핵심만, 더 보려면 위젯 우측상단 화살표 → 해당 페이지.
4. **섹션 단위 의미 그룹화**. 각 섹션은 투자자의 구체적 목적 하나를 달성.
5. **채팅은 플로팅 레이어**. 대시보드 그리드 안에 박지 않고, 모든 페이지 공통 플로팅 위젯으로 구현. 사용자가 위치·상태 제어.

## 플로팅 채팅 (FloatingChat)

대시보드 그리드와 분리된 **전역 플로팅 레이어**. `app/layout.tsx` 최상단에 삽입되어 모든 페이지에서 공통 노출.

### 상태 3가지

**① 닫힌 상태 (기본값 · 초기)**
- 좌하단 또는 우하단에 작은 원형 버튼만 노출 (56×56px)
- 아이콘: 💬 + unread 배지 (빨간 원 + 숫자)
- 배지 = 마지막 세션 종료 이후 새로 도착한 메시지 수

**② 최소화 상태**
- 가로 짧은 바 (240×40px)로 축소
- "마켓 채팅 · N명 접속 중" 표시
- 클릭하면 다시 열림

**③ 열린 상태**
- 너비 320px, 높이 440px (현재 ChatWidget과 유사한 크기 유지)
- 양 끝 여백 16px
- 헤더 / 메시지 리스트 / 입력창 구성

### 헤더 버튼 레이아웃

```
┌──────────────────────────────────┐
│ 💬 마켓 채팅   → 오른쪽    _  ×  │   ← 좌측에 있을 때
└──────────────────────────────────┘

┌──────────────────────────────────┐
│ 💬 마켓 채팅   ← 왼쪽      _  ×  │   ← 우측에 있을 때
└──────────────────────────────────┘
```

- **방향 아이콘 + 한글 라벨 조합** — `→ 오른쪽` / `← 왼쪽`. 아이콘만으로는 헷갈려서 텍스트로 "이동할 방향" 명시.
- `_` = 최소화 (위 ② 상태)
- `×` = 완전히 닫기 (위 ① 상태)

### 토글 동작

- 좌↔우 이동 시 `transition-all duration-300 ease-out` 슬라이드 애니메이션
- 위치 기억: `localStorage.chat_position` (`'left'` | `'right'`)
- 최소화 상태 기억: `localStorage.chat_minimized` (`boolean`)
- 첫 방문자는 **자동으로 열린 상태로 시작** (온보딩 효과)
- 두 번째 방문부터는 마지막 상태 복원

### 재참여 유도 장치 (retention)

1. **초회 자동 열림** — 신규/첫 세션 시 자동 확장
2. **Unread 빨간 배지 + 1회 깜빡임** — 새 메시지 도착 시 주의 끌기
3. **프리뷰 토스트** — 닫혀있을 때 메시지 도착하면 버튼 위 말풍선으로 2~3초 미리보기

### 모바일 대응

- 모바일 뷰포트(<md)에선 전체 화면 오버레이로 확장
- 좌우 토글 버튼은 숨김 (모바일에선 의미 없음)

**우측 "종목 상세" 컬럼 상세 (V3.2 확정)**

**레이아웃:**
- 스냅샷 헤더 고정 약 100px (상단)
- 탭 네비 바 (4개)
- 탭 콘텐츠 영역 (내부 스크롤)

**스냅샷 헤더 콘텐츠:**
- 종목명 · 코드
- 현재가 · 등락률
- 시 / 고 / 저 / 거래량 / 시총 / PER / 배당수익률

**탭 4개 (좌→우, 조회 빈도 순):**
1. **종합** (기본 선택)
2. **뉴스**
3. **공시**
4. **재무**

**"종합" 탭 콘텐츠 (위→아래, 조회 빈도 순):**
1. 핵심 투자지표 — PER · PBR · 시총 · 배당수익률 · 52주 신고·신저가
2. 투자자 수급 미니 🇰🇷 — 외인 / 기관 / 개인 당일·5일 순매수 (US 티커 선택 시 블록 숨김)
3. 뉴스 하이라이트 3건 (제목 + 시간 + 출처)
4. 공시 하이라이트 3건 (제목 + 시간)
5. 재무 미니 — 매출·영업이익 4분기 막대 (클릭 시 재무 탭으로 딥링크)

**"뉴스" 탭:** 해당 종목 전체 뉴스 리스트 (제목 / 시간 / 출처, 클릭 시 원문)  
**"공시" 탭:** DART(KR) / SEC(US) 전체 공시 리스트  
**"재무" 탭:** 손익 / 재무상태 / 현금흐름 상세 테이블

**설계 근거:**
- 탭 4개 순서 = 한국 개인투자자 조회 빈도 순 (네이버 증권 패턴 준거)
- "종합" 탭 블록 내부 순서도 동일 원칙
- 수급 미니를 종합 탭에 노출 = `/net-buy` 페이지 외에도 종목 상세에서 즉시 확인 수요 매우 높음
- 탭 구조 채택 = 단일 컬럼 내부 정보밀도 확보. 컬럼 간(좌·중·우) "한 화면 다층 관찰" 원칙은 그대로 유지.

**워크플로우**: 관심종목에서 종목 클릭 → 차트·호가·체결·종목상세가 **동시에 해당 종목으로 갱신**. 다음 종목 클릭 → 전환 스위치처럼 작동. 이 섹션 하나로 "종목 분석 워크스페이스" 완성.
