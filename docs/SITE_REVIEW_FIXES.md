### 2단계: 티커바 에러 아이콘 수정

```
상단 티커바(components/layout/TickerBar.tsx)에서 KOSPI, KOSDAQ, S&P500, DOW 옆에 빨간 에러 아이콘이 표시되고 있어.
이 지수들의 데이터를 가져오는 로직을 확인해줘.
TradingView Ticker Tape 위젯을 사용하고 있다면, symbols 설정이 올바른지 확인하고,
에러 아이콘이 나오지 않도록 수정해줘.
만약 커스텀 API로 지수 데이터를 가져오고 있다면, 에러 시 fallback 값을 표시하도록 해줘.
TradingView TickerTape 위젯 하나로 통합하는 것도 고려해줘.
```

### 3단계: 홈페이지 빈 상태(Empty State) UI 개선

```
홈페이지에서 공시, 뉴스, 수급 데이터가 없을 때 보여주는 empty state UI를 개선해줘.
현재 "오늘의 공시가 없습니다", "오늘의 뉴스가 없습니다"만 표시되는데, 이걸 더 보기 좋게 만들어줘:
1. 각 섹션에 적절한 아이콘 + 설명 텍스트 추가
2. "데이터가 곧 업데이트됩니다" 같은 안내 메시지
3. 수급 TOP 5 섹션(외국인/기관 순매수)도 빈 상태일 때 "데이터 준비 중" 표시
4. 배너 섹션(BannerSection)도 배너가 없을 때 "광고 문의: advertiser 페이지 링크" 안내
전체적으로 빈 상태여도 사이트가 완성도 있어 보이게 해줘.
```
