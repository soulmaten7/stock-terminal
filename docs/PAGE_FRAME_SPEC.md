# Stock Terminal — 전체 페이지 프레임 명세서
<!-- 2026-04-09 -->

## 설계 철학
- **라이브스코어 모델**: 홈 = 실시간 스코어보드 + 채팅방 (하루종일 붙어있는 곳)
- **시간축 스크롤**: 위 = "지금", 중간 = "오늘 전체", 아래 = "내일 예고"
- **4모니터 → 웹 페이지 변환**: 전업투자자 모니터 4대 구조를 페이지 단위로 분리
- **공통 기능은 기본 인프라, 차별화 기능이 메인 자리**: 어떤 증권사에나 있는 건 서브페이지로, 흩어진 정보를 모아주는 게 홈

---

## 전체 사이트맵

```
홈 (/)                          ← 라이브 스코어보드 + 채팅 (모니터4 역할)
종목 상세 (/stocks/[symbol])     ← 매매 판단 (모니터1+2 역할)
뉴스·공시 (/news)               ← 정보 피드 (모니터3 역할)
시장 분석 (/analysis)           ← 데일리 체크
스크리너 (/screener)            ← 조건검색
비교 분석 (/compare)            ← 종목 vs 종목
링크 허브 (/link-hub)           ← 외부 사이트 모음 (기존)
마이페이지 (/mypage)            ← 포트폴리오 + 알림 + AI분석
구독·결제 (/pricing)            ← 요금제 (기존)
관리자 (/admin)                 ← 내부 운영 (기존)
```


### 1층 (뷰포트 고정 영역) — "지금"

| 패널 | 내용 | 데이터 소스 | 업데이트 주기 |
|------|------|------------|-------------|
| 채팅 영역 | 실시간 대화, 200자 제한, 비속어 필터 | Supabase Realtime | 실시간 |
| 관심종목 현재가 | 등록 종목의 현재가/등락률/거래량, 숫자 깜빡임 | 한투 OpenAPI | 실시간 (장중) |
| 외국인/기관 순매수 | 실시간 순매수 상위 종목 | 한투 OpenAPI / KRX | 1분 간격 |
| 속보 뉴스/공시 | 최신 뉴스+공시 혼합 피드, 중요도 태그 | DART API + RSS | 5분 간격 |

### 2층 (스크롤) — "오늘 전체"

| 패널 | 내용 | 데이터 소스 | 업데이트 주기 |
|------|------|------------|-------------|
| 거래량 급등 종목 | 평소 대비 거래량 급증 종목 리스트 | 한투 OpenAPI / KRX | 5분 간격 |
| 프로그램 매매 동향 | 차익/비차익 거래 현황 | 한투 OpenAPI | 5분 간격 |
| 글로벌 선물 지수 | S&P500, 나스닥, 원유, 금, 달러 미니 카드 | TradingView 위젯 | 실시간 |
| 코스피/코스닥 미니 차트 | 당일 지수 흐름 차트 | TradingView 위젯 | 실시간 |
| 투자주의/경고 종목 | KRX 지정 주의/경고/위험 종목 | KRX | 장중 수시 |

### 3층 (스크롤) — "내일/이번 주"

| 패널 | 내용 | 데이터 소스 | 업데이트 주기 |
|------|------|------------|-------------|
| 경제지표 발표 일정 | CPI, 금리결정, 고용지표 등 일정 | FRED + 수동/Make | 1일 1회 |
| IPO 일정 | 신규 상장 예정, 공모가 밴드 | KRX + 수동 | 1일 1회 |
| 실적 발표 예정 | 이번 주 실적 발표 예정 종목 | DART + 수동 | 1일 1회 |
| 주요 이벤트 | 주주총회, 배당락일, FOMC 등 | 수동/Make | 1주 1회 |

### 광고 배치

**배너 크기**: 320px x 120px (가로형, 세로로 여러 개 쌓임)

| 위치 | 광고 등급 | 단가 | 대상 | 배너 수 |
|------|----------|------|------|--------|
| 1층 좌우 | 인증 광고 | 20일 5만원 | 증권사, 금융상품, 공식 리딩방 | 층당 4~5개 |
| 2~3층 좌우 | 일반 광고 | 20일 3만원 | 단톡방 링크, 주식 교육, 소규모 리딩방 | 층당 4~5개 |

**광고 클릭 옵션** (광고주가 배너 등록 시 선택):
- 옵션 A: 외부 링크로 바로 이동 (기본)
- 옵션 B: 플랫폼 내 광고주 전용 페이지 (/ad/[id])로 이동 (프리미엄)

### 광고주 전용 페이지 (/ad/[id])

광고주가 직접 꾸밀 수 있는 프로필 페이지:
- 로고/프로필 이미지, 업체명, 소개글
- 대표 이미지 (최대 5장)
- 외부 링크 목록 (카톡, 텔레그램, 홈페이지 등)
- 연락처, 운영시간, 태그
- 인증 광고주: 금색 뱃지 + 테두리

Supabase 테이블: ad_pages (advertiser_id, company_name, description, logo_url, images, links, tags, is_verified 등)

## 2. 종목 상세 (/stocks/[symbol]) — 매매 판단 페이지

**역할**: 모니터 1번(차트+호가) + 모니터 2번(종목감시+수급) 통합

```

│  종목명 + 현재가 + 등락률 + 거래량     [관심종목 추가 ★]  │
├────────────────────────┬────────────────────────────────┤
│                        │                                │
│   실시간 캔들차트       │    호가창 (10단)                │
│   (1분~월봉 선택)       │    ├─ 매도 10단계               │
│   보조지표 토글         │    ├─ 현재가                    │
│   (이평선/MACD/RSI/     │    └─ 매수 10단계               │
│    볼린저/거래량)       │                                │
│                        │    체결 내역                     │
│                        │    (실시간 틱 데이터)            │
│                        │                                │
├────────────────────────┴────────────────────────────────┤
│  탭 네비게이션                                           │
│  [수급] [재무] [공시] [뉴스] [공매도/신용] [배당] [AI분석]  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  선택된 탭 콘텐츠                                        │
│                                                         │
│  [수급 탭 예시]                                          │
│  외국인/기관/개인 일별 순매수 차트                         │
│  + 당일 실시간 순매수 테이블                              │
│                                                         │
│  [재무 탭 예시]                                          │
│  PER/PBR/EPS/ROE + 연간/분기별 매출/영업이익/순이익       │

├─────────────────────────────────────────────────────────┤

```

### 데이터 소스 매핑

| 섹션 | 데이터 소스 | 한투 API 의존 |
|------|------------|-------------|
| 캔들차트 | TradingView Advanced Chart | X (위젯) |
| 호가창 | 한투 OpenAPI 실시간 | O |
| 체결내역 | 한투 OpenAPI 실시간 | O |
| 수급 | 한투 OpenAPI / KRX | O |
| 재무제표 | DART API → financials 테이블 | X |
| 공시 | DART API → disclosures 테이블 | X |
| 뉴스 | RSS → news 테이블 | X |
| 공매도/신용 | KRX → short_credit 테이블 | X |
| 배당 | DART API → dividends 테이블 | X |
| AI분석 | OpenAI GPT → ai_analysis 테이블 | X |

## 6. 비교 분석 (/compare) — 종목 비교 페이지

## 구현 우선순위

### Phase 1 — 홈 리팩토링 (한투 API 전에 가능)
1. HomeClient.tsx 3층 구조로 재설계

3. 1층: 채팅(기존) + 라이브 스코어보드(관심종목 테이블 + 속보 피드)

5. 3층: 경제 캘린더 (FRED 연동) + 일정 카드

### Phase 2 — 한투 API 연동 후
6. 1층 관심종목에 실시간 가격 연결

8. 2층 거래량 급등 종목
9. 2층 프로그램 매매 동향

### Phase 3 — 신규 페이지
11. 뉴스·공시 전체 페이지 (/news)
12. 시장 분석 페이지 (/analysis)
13. 스크리너 페이지 (/screener)
