## 🚀 사용법

### 방법 A — 업종 표준본 그대로 발송 (가장 빠름)

광고주가 이미 있는 업종이면 해당 PDF를 카톡·문자·이메일에 바로 첨부.
예: 개인회생 광고주 → `01_개인회생.pdf` 그대로 전송.

### 방법 B — 광고주 회사명 넣은 맞춤 PDF 생성 (30초)

터미널에서:
```bash
cd /sessions/admiring-modest-johnson/mnt/OTMarketing/templates/proposals
python _generator.py --industry debt-relief --company "뉴스타트브릿지"
```

결과: `OT_MARKETING_운영제안서_뉴스타트브릿지.pdf` 생성됨.

**업종 키 목록:**
- `debt-relief` (개인회생)
- `rental-water` (정수기렌탈)
- `broadband` (인터넷통신)
- `invest-lead` (주식리딩)
- `realestate` (부동산분양)
- `medical` (병의원)

### 방법 C — 특수 문구가 필요한 경우 (수작업)

1. `_MASTER_운영제안서.docx` 파일을 Word로 열기
2. `Ctrl+H` (찾아 바꾸기) 로 `{변수명}` 교체:
   - `{회사명}` → 광고주 회사명
   - `{업종_설명}` → "개인회생 CPA 파트너십 제안" 등
   - `{업종_상세_이름}` → "개인회생·파산면책" 등
   - `{집행_매체_리스트}` → "메타 / 당근 / 네이버 / 유튜브"
   - `{리드_정의_1~3}` → 업종별 리드 정의 3줄
   - `{발송일}` → "2026-04-23" 형식
   - `{연락처}` → "010-XXXX-XXXX"
3. Word에서 PDF로 저장 (파일 → 다른 이름 저장 → PDF 선택)

## ➕ 새 업종 추가하기

`_generator.py` 파일 상단의 `INDUSTRIES` 딕셔너리에 항목 추가:

```python
"new-industry-key": {
    "idx": "07",
    "name_kr": "새 업종 이름",
    "subtitle": "새 업종 CPA 파트너십 제안",
    "lead_def": [
        "리드 정의 1번 줄",
        "리드 정의 2번 줄",
        "리드 정의 3번 줄",
    ],
    "channels": "집행 매체 리스트",
},
```

그 다음 아래 `CLEAN_NAMES` 에도 한글 파일명 추가:
```python
"new-industry-key": "새업종",
```

저장 후 `python _generator.py --all` 실행하면 자동 생성.
